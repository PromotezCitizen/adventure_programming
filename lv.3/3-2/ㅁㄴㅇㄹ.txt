사용법
1. load를 눌러 파일 불러오기
	save 버튼이 활성화되어있다면 save 버튼 비활성화
	1.1 파일을 불러온 경우 convert 버튼 활성화
	1.2 파일을 불러오지 못한 경우 convert 버튼 비활성화 유지
2. 1.1이 성공한 경우 convert 버튼 클릭
	이 경우 다시 변환하는 과정을 거치지 않도록 convert 버튼 비활성화
	save 버튼 활성화
3. save 버튼이 활성화되면 save 버튼 클릭
	3.1 저장이 된 경우 다시 저장하지 않도록 save 버튼 비활성화
	3.2 취소 버튼을 눌러서 저장을 하지 않은 경우에는 save 버튼 활성화 유지
	
load 버튼은 언제든 누를 수 있다
부호화된 파일인지 부호화되지 않은 파일인지 자동으로 확인

해당 창은 MODAL 방식으로 되어있습니다
